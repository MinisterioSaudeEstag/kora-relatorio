# Chat page

## Objective
Permitir interação entre usuário e sistema de IA para construção de relatórios.

## Features
- Caixa de texto para perguntas.
- Exibição das respostas da IA.
- Histórico de interações.

## Flow
1. Usuário acessa `/chat`.
2. Digita pergunta na caixa de texto.
3. Sistema processa e retorna resposta.
4. Histórico é atualizado.

## Considerations
- Interface simples e clara.
- Possibilidade de expandir para múltiplos documentos.
