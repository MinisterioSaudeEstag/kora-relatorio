# Config page

## Objective
Permitir que o usuário personalize sua experiência e ajuste preferências do sistema.

## Features
- Alterar senha.
- Configurar preferências de idioma e tema (dark/light).
- Gerenciar notificações.
- Opção de excluir conta.

## Flow
1. Usuário acessa `/configuracoes`.
2. Dados atuais são carregados do banco.
3. Usuário altera preferências.
4. Alterações são salvas e aplicadas imediatamente.

## Considerations
- Garantir segurança ao alterar senha.
- Validar dados antes de salvar.
- Interface clara e responsiva.
