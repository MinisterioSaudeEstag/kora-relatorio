# Specification (Spec)

## Main Functionality
- Upload de PDF
- Caixa de texto para perguntas
- Resposta baseada no conteúdo do PDF

## Functional Requirements
- Autenticação JWT
- Interface responsiva
- Armazenamento seguro dos documentos
- Histórico de perguntas e respostas

## Non-Functional Requirements
- Escalabilidade
- Segurança
- Performance
